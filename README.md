# A/B Test Statistical Analysis

## 📊 Project Overview
Statistical analysis of a 10,000-user A/B test to evaluate the impact of a new website feature on user conversion rates.

## 🎯 Key Findings
- **2.48% absolute conversion lift** from the new feature
- **24.1% relative improvement** over control group  
- **Statistically significant** result (p < 0.0001)
- **Potential 24% revenue increase** based on conversion improvements

## 🛠️ Tools & Technologies
- **Python**: Pandas, NumPy, SciPy, Matplotlib, Seaborn
- **SQL**: Database design, queries, user segmentation
- **Statistics**: Two-proportion z-test, hypothesis testing, confidence intervals

## 📁 Project Structure
- `ab_test_analysis.ipynb` - Main analysis notebook
- `ab_test_data.csv` - Test data (10,000 users)
- `ab_test.db` - SQLite database
- `queries.sql` - SQL analysis queries

## 📈 Methodology
1. Data generation and database setup
2. Exploratory data analysis
3. Statistical hypothesis testing
4. Device-based segmentation analysis
5. Business recommendations

## 💡 Business Impact
Based on the analysis, implementing the new feature could increase revenue by 24% through improved conversion rates, particularly among mobile users.
